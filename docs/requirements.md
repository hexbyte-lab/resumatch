# ResuMatch - Requirements

## Project Description

ResuMatch is a hybrid web application that helps job seekers tailor their resumes to specific job postings. Users upload both their resume and a job description, and the system analyzes the text to provide a match score and improvement suggestions.

## Core Features (MVP)

1. Upload or paste resume text.
2. Upload or paste job description text.
3. Compare texts to compute a match score.
4. Identify missing relevant keywords.
5. Provide suggestions for improving resume alignment.

## Input/Output

**Input:**

* Resume text
* Job description text

**Output:**

* Match score (0-100%)
* List of missing skills/keywords
* Suggested resume improvements
